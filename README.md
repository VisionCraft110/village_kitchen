🍲 Village Kitchen – Restaurant Website

Village Kitchen is a responsive and user-friendly restaurant website designed to showcase traditional and authentic cuisine. The website highlights the restaurant’s menu, services, and contact information with an attractive and modern interface.

🌟 Features

🏠 Home page with restaurant introduction

📖 Complete menu section (Veg & Non-Veg items)

🛒 Cart functionality for adding items

📱 Fully responsive design (Mobile & Desktop friendly)

🎨 Clean and modern UI layout

📍 Contact and location details

🛠️ Technologies Used

HTML5

CSS3

JavaScript

├── index.html

🎯 Project Objective

This project was developed to create a realistic restaurant website that:

Displays menu items clearly

Provides a smooth user experience

Demonstrates frontend web development skills

Can be customized for real-world restaurant use

🚀 How to Run the Project

Download or clone the repository

Open the project folder

Double-click on index.html

View the website in your browser
